# Matriks RACI

Keterangan:

R - Responsible | Penanggung Jawab / Pelaksana

A - Accountable | Akuntabel / Pengambil keputusan

C - Consulted | Dikonsultasikan / Penasihat

I - Informed | Diinformasikan / Penerima Laporan
